# 多線程下的單例模式

## 餓漢式(不會造成多線程安全問題)
```
public class Single {

    private static final Single s = new Single();
    private Single() {
    
    }
    
    public static Single getInstance() {
    
        return s;
    }
}
```

## 懶漢式(會造成安全問題)
```
public class Single {
	
	private static Single s = null;
	
	private Single() {
		
	}
	
	public static Single getInstance() {
		
		if(s == null) {//雙重判斷，提高效率
			
			synchronized(Single.class) {//同步鎖，解決安全問題
										//非靜態括號中可用 this.getClass()，但靜態中不可。
				
				if(s == null) {
				
					s = new Single();
				}
			}
		}
		
		return s;
	}
}
```
