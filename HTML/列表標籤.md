# 列表標籤

## 使用
- 列表標籤: dl
 - 上層項目: dt
 - 下層項目: dd -> 封裝的內容是有**自動縮進效果**的。
- 有序和無序的項目列表
 - 有序: ol 標籤
 - 無序: ul 標籤
 - 無論有序和無序，條目的封裝用的都是**li 標籤**，而且他們都有縮進效果。

## Example
```
<!DOCTYPE html>
<html>
<head>
	<title>列表標籤</title>
</head>
<body>
	<h1>New Web Project</h1>

	<!--html 列表標籤-->
	<dl>
		<dt>上層項目</dt>
		<dd>下層項目</dd>
		<dd>下層項目</dd>
		<dd>下層項目</dd>
	</dl>
	<!--有序和無序的項目列表-->
	<ul type="square">
		<li>無序項目列表</li>	
		<li>無序項目列表</li>	
		<li>無序項目列表</li>	
		<li>無序項目列表</li>	
	</ul>
	<ol type="a">
		<li>有序項目列表</li>
		<li>有序項目列表</li>
		<li>有序項目列表</li>
		<li>有序項目列表</li>		
	</ol>
</body>
</html>
```
